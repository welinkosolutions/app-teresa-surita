# APP Teresa Surita

Repositório oficial do projeto **APP Teresa Surita**.

Este projeto será construído do zero, com documentação contínua, separação clara de responsabilidades e foco em segurança, rastreabilidade e evolução modular.

## Estado atual

- Projeto em fase inicial de alinhamento.
- Diretrizes técnicas e estratégicas em definição.
- Nenhuma stack, banco de dados ou arquitetura foi assumida como definitiva.

## Princípio central

O APP Teresa Surita deve ser tratado como uma plataforma de relacionamento, comunicação, escuta ativa, mobilização e gestão de demandas da população.

## Regras de condução

1. Não misturar este projeto com outros projetos.
2. Não inventar regras, dados, integrações ou decisões técnicas.
3. Documentar decisões antes de implementar.
4. Priorizar segurança desde o início.
5. Evitar remendos e soluções temporárias sem registro formal.
