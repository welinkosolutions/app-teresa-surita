# Diretrizes do Projeto APP Teresa Surita

## Objetivo
Criar uma plataforma de relacionamento entre equipe e população.

## Regras Fundamentais

1. Não misturar projetos.
2. Não assumir requisitos sem validação.
3. Documentar decisões.
4. Segurança desde a concepção.
5. Evitar dívida técnica desnecessária.
6. Mobile first.
7. Arquitetura modular.
8. Histórico auditável.
9. Pessoa como entidade principal.
10. Métricas desde o início.
